Common Solutions

Restore Purchases

Open SavvyScan → Account → Restore Purchases. Use the same Apple ID you bought Premium with.

Cancel Subscription

Settings → Apple ID → Subscriptions → SavvyScan → Cancel. Apple manages billing/refunds.

Common Support Issues

Scans not unlocking

Try Restore Purchases and restart the app. Make sure you're signed in with the same Apple ID you used to purchase Premium.

Scan limit reached

Wait for daily reset or watch a rewarded ad if available. Premium subscribers get unlimited scans.

Price mismatch

Amazon prices change frequently; refresh to get the latest. Prices are pulled in real-time from Amazon's API.
