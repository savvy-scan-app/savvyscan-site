Common Issues

How to Restore Purchases

Open SavvyScan → Account → Restore Purchases. Use the same Apple ID you bought Premium with.

How to Cancel Your Subscription

Settings → Apple ID → Subscriptions → SavvyScan → Cancel. Apple manages billing/refunds.

Frequently Asked Questions

Scans not unlocking

Try using the Restore Purchases feature (Account → Restore Purchases) and restart the app. Make sure you're signed in with the same Apple ID you used to purchase Premium.

Scan limit reached

Wait for the daily reset (usually at midnight) or watch a rewarded ad if available in your region. Premium subscribers get unlimited scans.

Price mismatch

Amazon prices change frequently throughout the day. Pull down to refresh and get the latest pricing information. Prices shown are from Amazon's live API.
